# 3、设计实验
## 3.1 实验目标是什么
- 证明H1：AI 在训练分布内能快速逼近 AC 潮流解，但在拓扑变化和临界工况下物理残差显著增大
- 证明H2：使用 AI 预测作为 Newton 初值，可以保持传统潮流的最终精度，同时降低平均迭代次数或提高困难工况的收敛率
## 3.2 实验组如何设计
| 组别 | 含义 | 用途 |
|---|---|---|
| NR+平启动 | 传统牛拉法+初值采用零 | 标准冷启动对照 |
| NR+上一时刻解 | 传统牛拉法+初值采用上一时刻解 | 生产热启动强基线 |
| PQ法+平/热启动 |---| 传统快速算法基线 |
| 纯AI | --- | 代理模型速度与误差 |
| AI 初值 + 全 NR | --- | 高精度混合方法 |
| AI 初值 + PQ法 |---| 可选的快速混合方法 |

参考解建议用：
全 AC Newton–Raphson + 严格功率失配阈值，并抽样与另一成熟求解器交叉验证。

最终重点比较：最终功率失配；收敛率；平均和最坏迭代次数；p50/p95/p99 总延迟；电压与线路越限判断；正常、重载、拓扑变化下的表现；AI 使传统算法变慢或不收敛的比例。

最关键的研究问题不是“AI 是否比平启动 NR 快”，而是：
AI 初值能否稳定优于上一时刻解和快速解耦等成熟低成本方法，同时由传统求解器保证最终精度？
