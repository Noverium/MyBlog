## 结论

两种算法均在设定阈值内收敛，且满足系统有功功率平衡。FDXB 与 NR 的最大电压幅值差为 $(1.31\times10^{-10}) p.u.，最大相角差为 (2.59\times10^{-8})°$，说明二者在该正常基准工况下收敛至几乎相同的 AC 潮流解。

FDXB 的迭代次数高于 NR，但 IEEE 14 系统规模过小，单次 Python 计时受软件调用和数据处理开销影响较大，不能据此得出两种算法实际计算速度的普遍结论。
